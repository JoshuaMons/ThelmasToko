# Thelma's Toko

A modern one-page website for [Thelma's Toko](https://www.instagram.com/thelmas.toko/) — homemade food and good eats. Menu, specials, and orders are shared via Instagram.

## Contents

- **index.html** — Single-page site with hero, about, menu, and contact
- **styles.css** — Responsive layout and styling

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
