# Thelma's Toko

A modern one-page website for [Thelma's Toko](https://www.instagram.com/thelmas.toko/) — homemade food and good eats. Menu, specials, and orders are shared via Instagram.

## Contents

- **index.html** — Single-page site with hero, about, menu, and contact
- **styles.css** — Responsive layout and styling

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Push to GitHub

1. Create a new repository on [GitHub](https://github.com/new) (e.g. name: `thelmas-toko-website`, leave it empty — no README).
2. In this folder, run (replace `YOUR_USERNAME` with your GitHub username):

```bash
git remote add origin https://github.com/YOUR_USERNAME/thelmas-toko-website.git
git branch -M main
git push -u origin main
```

If your repo already uses `master`, use `git push -u origin master` instead.
